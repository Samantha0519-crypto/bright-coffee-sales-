# bright-coffee-sales-
Bright Coffee Shop Sales Analysis
Overview

This project was completed as part of the BrightLearn Data Analytics Case Study. The objective was to analyse transactional coffee shop sales data and provide insights that could help management improve product performance and increase revenue.

The project follows a complete data analytics workflow, beginning with project planning and data preparation, followed by SQL analysis, dashboard development in Microsoft Excel, and business recommendations.

Business Problem

Bright Coffee Shop recently appointed a new CEO with the goal of increasing revenue and improving product performance. As the Junior Data Analyst, my responsibility was to analyse historical sales data and identify actionable insights that support business decision-making.

The analysis focuses on answering questions such as:

Which products generate the most revenue?
What time of day generates the highest sales?
Which product categories perform best?
How can sales performance be improved?
Project Objectives
Explore and understand the dataset
Clean and transform transactional data using SQL
Create new business metrics through feature engineering
Analyse sales performance across products and time periods
Build an interactive Excel dashboard
Present findings and recommendations supported by data

Key Findings

The analysis provides insights into:

The highest-performing product categories
Revenue generated across different stores
Peak sales periods throughout the day
Product performance
Overall revenue trends

These insights can assist management with inventory planning, staffing decisions, marketing campaigns, and product strategy.

Recommendations

Based on the analysis, the following recommendations can support business growth:

Increase stock levels for top-performing products.
Schedule more staff during peak trading periods.
Promote underperforming products through targeted campaigns.
Monitor sales trends regularly using an automated dashboard.
Continue tracking store performance to support future expansion.
